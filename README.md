This project presents an end-to-end E-Commerce Business Intelligence analysis using SQL to extract, transform, and analyze transactional data. It focuses on key business metrics such as revenue performance, customer segmentation, product performance, regional sales distribution, and time-based growth trends. Advanced SQL techniques including CTEs, window functions, aggregations, and subqueries are used to generate actionable insights. The project simulates real-world business scenarios and demonstrates how structured SQL analysis can support strategic decision-making, improve customer understanding, and optimize overall business performance.
